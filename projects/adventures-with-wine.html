Adventures with Wine

Occasionally, you need to run Windows programs even if you’re a die-hard Linux fanatic like myself. Part of the reason I decided to convert to Linux full time was that I genuinely enjoy solving strange computational problems which come up from time to time when running Linux. My preferred mode to run Windows programs is to use WINE. As WINE is not a perfect implementation of Windows, occasionally programs fail to build or run properly.

Since I love fixing these types of problems, I will sometimes post my experience solving WINE-related problems. I hope they help other would-be WINE developers understand how they might solve their own problems! Also I think some issues I deal with could be interesting to the developer not necessarily interested in WINE either!

Posts having to do with WINE will be labelled with an ‘Adventures with Wine’ so the reader will know what they are getting themselves into!
